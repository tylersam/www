environment

I've used a variety of tools and environments over the years and I feel like I'm still looking for the ideal setup. I've bounced between macOS, Windows, and Ubunutu using a variety of editors along the way — Sublime, Atom, RubyMine, VSCode.

I've also tried a series of approaches for deploying and hosting my apps. I've used Heroku, Capistrano + semi-managed VPS, Cloud66 + Digital Ocean, and Hatchbox + DO. Fortunately, this is an area where I feel like I've found the right solution for Rails apps: Hatchbox. It's a fantastic service that makes it simple (and affordable) to deploy and manage apps.

Currently, I work on a PC (because I like building computers) and I type on an Ergodox EZ with Cherry MX blue switches. This keyboard is my single favorite piece of hardware — it's incredible to type on and it has absolutely saved my wrists!

My text editor of choice right now is VS Code with the Vim extension. I use Prettier to auto-format my Ruby and JS.

My environment is currently:
  • Ergodox EZ with Cherry MX Blue switches
  • Custom built PC
  • WSL 2 running Ubuntu 20.04
  • asdf-vm for managing runtime versions
  • Hatchbox for deploys
  • Digital Ocean
  • VS Code + Vim extension
  • Prettier for Ruby
