How I Build
How I Work

I started building Rails apps after reading the Ruby on Rails Tutorial by Michael Hartl back in 2013. I felt productive immediately, convention over configuration was freeing, the number of resources available via gems or blog posts was empowering, and the number of complexities the framework handles out of the box was incredible.

As a solo developer, Rails simply provides remarkable productivity. I've mostly stuck with a vanilla Rails stack with some React components here and there. However, tools like Stimulus Reflex are opening the door for greater levels of interactivity without adding in adding the complexity and duplicated logic of a full client side framework. I'm excited to see how it and the tools Basecamp is extracting from Hey will shape Rails apps in the next few years.

Ruby is also a major reason I continue to use Rails - it's clean, expressive, and simply enjoyable to write. Ruby is designed to make programmers happy, and after all these years, it still makes me happy.

My standard stack currently starts with:
  • Rails
  • PostgreSQL
  • Webpacker
  • Stimulus (+Reflex)
  • Turbolinks
  • Bulma CSS
  • Delayed Job
  • Devise
  • RSpec, Capybara, Factory Bot
  • React Native (mobile apps or TV apps)

I've used a variety of tools and environments over the years, and I still feel like I'm looking for the ideal setup. I've bounced between MacOS, Windows, and Ubunutu using a variety of editors along the way - Sublime, Atom, RubyMine, VSCode.

I've also tried a series of approaches for deploying and hosting my apps. I've used Heroku, Capistrano + semi-managed VPS, Cloud66 + Digital Ocean, and Hatchbox + DO. Fortunately, this is an area where I feel like I've found the right solution for Rails app: Hatchbox. It's simply a fantastic service that makes is simple (and affordable) to deploy and manage my apps.

Currently, I work on a PC (because I like building computers) and I type on an Ergodox EZ with Cherry MX blue switches. This keyboard is my single favorite piece of hardware I own - it's incredible to type on and it has absolutely saved my wrists!

My text editor of choice right now is VS Code with the Vim extension. I use Prettier to auto format my Ruby and JS.

My environment is currently:
  • Ergodox EZ with Cherry MX Blue switches
  • Custom built PC
  • WSL 2 running Ubuntu 20.04
  • asdf-vm for managing runtime versions
  • Hatchbox for deploys
  • Digital Ocean
  • VS Code + Vim extension
  • Prettier for Ruby
