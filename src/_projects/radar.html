Food Cost Management

Radar is an internal tool used to manage recipes, inventory, and food costs for our restaurants. It is used for building recipes, tracking product mix, monitoring purchasing, taking inventory, and creating action plans for resolving food cost issues.

It's the central tool used to track and control our food cost. It started out of a need to handle poor data coming out of the point of sale system and has developed into a tool built around organization specific food management processes.

Tech Stack
Ruby on Rails PostgreSQL Redis React

Radar is essentially a vanilla Ruby on Rails app that uses Postgres, Redis, and S3 for data storage and uses primarily server rendered HTML with React (via Webpacker) Components mixed throughout for the frontend.
