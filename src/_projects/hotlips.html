Customer Feedback

This project is an easy to use customer survey tool that collects feedback specific to the company's service model. It displays views of the data that have been integrated with point of sale data as well as customer loyalty data in the central internal system.

Tech Stack
Ruby on Rails PostgreSQL Stimulus JS

This project is a vanilla Ruby on Rails 6 app - PostgreSQL, Webpacker, Turbolinks, Stimulus JS, and RSpec.
