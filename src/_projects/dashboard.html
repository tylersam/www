Although I started this project as a tool to automate a single manual reporting task, it has grown into the central location for performance metrics for the company.

This app aggregates and provides views of data from a dozen sources — sales, labor costs, food costs, order times, social media reviews, weather, and more — to provide timely access to information as well as automate manual processes.
