The Food Cost Management system is used for building recipes, tracking product mix, monitoring purchasing, taking inventory, and creating action plans. The project began out of a need to handle messy data from the point of sale system and has developed into a tool built around organization-specific processes. With the help of the company's executive chef, we built this tool to track and control food costs.

Tech Stack
{% for tag in page.tags %} {% assign tag_class = tag | downcase | split: " " | join: "-" %} {{ tag }} {% endfor %}

This project is a Ruby on Rails app that uses Postgres, Redis, and S3 for data storage and uses a mixture of server rendered HTML and React components (via Webpacker) for the frontend.
