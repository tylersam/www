Meal 4 Meal Donation Tracking

For every meal purchased at Tacos 4 Life, a meal is donated through Feed My Starving Children. Each store displays a continuously updating meal count helping guests connect with the company mission to fight world hunger. The marketing team also uses the system to showcase current initiatives and promotions.

Tech Stack
React Native Apple TV Ruby on Rails

The frontend of this project is an Apple TV App distributed through the company's MDM. I built the app using React Native with a Ruby on Rails backend. Apple TVs allow for simple company-wide managment and offer a user-friendly experience for store leaders.
